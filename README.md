# 🗡️ 魂魄妖梦的次元收容所 (Youmu's Dimensional Clipboard)

> *"此剑，能斩断现世之迷惘；此卷，可铭记万物之须臾！"*

欢迎来到**半人半灵的庭师**——魂魄妖梦所守护的绝对领域（在线剪切板）。在这个被遗忘于白玉楼深处的次元裂缝中，您可以安全地封印那些无处安放的文字与幻影（图片）。

## 🔮 禁忌渊源 (项目简介)

这是一个基于 Java + Vue3 (Vite) + SQLite 构建的法术阵地，用于在跨越不同次元（设备终端）时，实现灵魂碎片（文本/图片）的极速量子纠缠与无缝同步。只需将其部署于服务器之上，即可随时随地召唤封印其中的知识与影像。

## 📜 固有结界能力 (核心特性)

- 🗡️ **六道轮回之境 (六宫格文本面板)**：突破常理的六个独立封印阵，支持同时束缚六段不同的言灵（文本数据），满足多线作业的魔法需求。
- 👻 **半灵的幻影视界 (图片剪切板)**：不只是文字，即使是稍纵即逝的视觉幻象（图片），也能在此被完美定格与提取。无图状态下亦有优雅的结界提示。
- ⏳ **阿卡夏记录回溯 (历史快照显示)**：铭刻每一次世界线的微小变动。你所修改的每一次命运（历史编辑记录），都能在时空缝隙中全知全能地展示。
- 🛡️ **永恒冥界的基石 (SQLite 持久化)**：摆脱了脆弱的现世内存态，数据的灵魂已被永久刻印在 SQLite 的不朽石碑上，纵使天地崩毁（服务器重启）亦不生不灭。
- 🌸 **白玉楼的春之律动 (妖梦主题 UI)**：采用了符合现代大结界美学的视觉语言。浅绿色渐变背景与灵动的交互布局，在每次点击中都能感受到庭师挥剑的干脆与西行妖的微风。

## ⚙️ 灵力刻印仪式 (环境与启动)

要唤醒这个法阵，你需要具备一定的灵力基础：后端需信奉 **Java (JDK 17+)**，前端需掌控 **Node.js (Yarn)** 的古老魔术。

### 1. 启动结界核心 (Backend)

一切灵力的源泉，数据库会在初次降临时自动生成创世石板 (`clipboard.db`)。

```bash
# 先锻造可供降灵的法器
mvn -DskipTests package

# 若你只是独自守望这座结界，此为节制灵压的推荐刻印
java \
  -Xms16m \
  -Xmx80m \
  -Xss256k \
  -XX:MaxMetaspaceSize=64m \
  -XX:ReservedCodeCacheSize=32m \
  -XX:+UseSerialGC \
  -jar target/YoumuClipboard-1.0.jar
```

若你只是以一人之力维持这片白玉楼偏殿中的小型法阵，上述灵压刻印已足以支撑日常运转。
但请谨记：幻影通道仍受 `clipboard.picture.max-size` 约束，其默认上限为 `10MB`。
一旦你试图放行更为庞大的影像残响，便应同步提高 JVM 堆的容量，否则结界将在瞬时波动中更容易失衡。

### 2. 咏唱幻象真言 (Frontend)

将数据转化为肉眼可见的华丽视界：

```bash
cd clipboard-web

# 汲取以太源力，重构世界树依赖包
yarn install 

# 唤醒白玉楼的前端幻象，映射至现世浏览器
yarn dev
```

- **视觉幻象重构 (Frontend)**: Vite 异星科技, Vue3, 以及注入了魂魄家灵魂的魔改极意 CSS。
